# Cloudmersive (cloudmersive)

Cloudmersive provides a portfolio of utility APIs covering virus and malware scanning, document conversion, OCR, image recognition, NLP, validation, security threat detection (spam, phishing, fraud, DLP, CDR), speech, video, barcode, currency, configuration, and data integration. Each API ships with a Swagger 2.0 / OpenAPI specification and SDKs in many languages, and is consumable on `api.cloudmersive.com` behind an API key (`Apikey` header).

**APIs.json:** [apis.yml](https://raw.githubusercontent.com/api-evangelist/cloudmersive/refs/heads/main/apis.yml)

## APIs

### Cloudmersive Virus Scan API
Scan files and content for viruses, malware, executables, scripts, macros, password-protected files, and other threats. Includes basic and advanced multi-engine scan modes plus website and cloud-storage scanning (Azure Blob, S3, GCS, SharePoint Online).

- [Documentation](https://api-console.cloudmersive.com/swagger/index.html?urls.primaryName=Virus%20Scan%20API)
- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/virus)
- [OpenAPI (local)](openapi/cloudmersive-virus-scan-openapi.json)

### Cloudmersive Security Threat Detection API
Detect SQL injection, XSS, XXE, SSRF, command injection, and other content-borne attacks.

- [Documentation](https://api-console.cloudmersive.com/swagger/index.html?urls.primaryName=Security%20API)
- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/security)

### Cloudmersive Spam Detection API
AI-powered spam detection for email and message content.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/spam)

### Cloudmersive Phishing Detection API
Detect phishing in email content and URLs.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/phishing)

### Cloudmersive Content Disarm and Reconstruction (CDR) API
Sanitize user documents by stripping macros, scripts, and other embedded threats.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/cdr)

### Cloudmersive Fraud Detection API
Document fraud and content security threat scanning.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/fraud)

### Cloudmersive Data Loss Prevention (DLP) API
Detect and redact PII and other sensitive data.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/dlp)

### Cloudmersive Document Convert API
Convert between many file formats (DOCX, PDF, HTML, XLSX, PPTX, CSV, JSON, XML), URL screenshots, document editing, tabular processing.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/convert)

### Cloudmersive Barcode API
Generate and recognize barcodes (QR, EAN, UPC, Code 128, etc.).

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/barcode)

### Cloudmersive Image Recognition and Processing API
Image classification, object detection, NSFW detection, face detection, editing, filters, and resizing.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/image)

### Cloudmersive NLP API
Tokenization, POS tagging, sentence splitting, language detection, translation, sentiment analysis, and rephrasing (v1 and v2).

- [OpenAPI v1](https://api-console.cloudmersive.com/swagger/api/nlp)
- [OpenAPI v2](https://api-console.cloudmersive.com/swagger/api/nlpv2)

### Cloudmersive OCR API
Deep-learning OCR for images and PDFs with form, receipt, and business-card extraction.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/ocr)

### Cloudmersive Speech API
Speech-to-text and text-to-speech in multiple languages.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/speech)

### Cloudmersive Validate API
Validate emails, phone numbers, domains, IP addresses, addresses, and other inputs.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/validate)

### Cloudmersive Video API
Convert, edit, and process video and audio.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/video)

### Cloudmersive Currency API
Real-time fiat and crypto currency exchange rates.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/currency)

### Cloudmersive Configuration API
Hosted configuration management and feature flags.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/config)

### Cloudmersive Data Integration API
Connect, transform, and integrate data across systems and file formats.

- [OpenAPI](https://api-console.cloudmersive.com/swagger/api/dataintegration)
